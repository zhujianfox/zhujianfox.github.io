HEXO常用命令及配置

发表于 2018-05-19 | 分类于 工具使用

这里会持续总结一些HEXO的使用方法，不定期更新！

Hexo 是一个快速、简洁且高效的博客框架。

常用命令

1.hexo s

打开hexo服务 打开后本地可以访问http://localhost:4000

2.hexo g

生成静态文件

3.hexo d

部署网站到GitHub 2、3两条命令可以合起来用 hexo g -d或者 hexo d -g

4.hexo clean

清除缓存文件 (db.json) 和已生成的静态文件 (public)。在某些情况（尤其是更换主题后），如果发现您对站点的更改无论如何也不生效，您可能需要运行该命令。

5.hexo n 文章名

创建新的文章
